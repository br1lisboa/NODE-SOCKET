# SOCKET

## *Installation*
*Clone the repository.* 

*Run in console:*

*1-```npm install``` (to install dependencies)*

*2-Create your own .env file with ur PORT.*

*3-```node app.js``` (to start the application)*

## *Some characteristics*
*Socket fundamentals.*

*Particularly:*
* Detect user connections and disconnections.
* Issue client server / client server messages.
* Listen to messages server client / client server.
* Broadcast.
* Callbacks on sockets.
